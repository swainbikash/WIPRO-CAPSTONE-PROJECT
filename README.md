## Description Of The Project
This project is a Linux Terminal based File Explorer created using C++ Language .  
It allows the user to perform different file and directory operations directly from console using a menu driven interface.

It supports basic file operations along with some extra (novelty) features.

### Basic Features
- List files & folders
- Enter directory
- Go back (parent directory)
- Create file
- Delete file
- Copy file
- Move / Rename file
- Search files (recursive)
- Show file permissions
- Change file permissions
- Open and view file content
